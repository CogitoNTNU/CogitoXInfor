## Prerequisite to run the project:

- Node v20 or newer
- Docker (desktop)
- Python v3.10 or newer
- Git LFS

## Run project:

### Backend (must have docker running):

```bash
cd backend
docker-compose up --build
```

### Frontend:

```bash
cd frontend
npm i
npm run dev
```
